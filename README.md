## ✉️ Git Commit Message Convention ✉️ 

### 🌿 Branch
main : 서비스 운영 브랜치 </br>
dev : 개발중인 브랜치 </br>
fe : 프론트 브랜치 </br>
be : 백엔드 브랜치 </br>
fe-feat(또는 be)/기능명 : 기능 개발 브랜치 </br>

### 💌 Commit Message
|Tag|Title|
|:---:|:---|
